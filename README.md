# Prism Theme One Dark

Theme based on Atom Dark One for Prism JS.

## Install

1. Install [prismJS](https://github.com/PrismJS/prism)
2. Add `prism-onedark.css` to your project
3. change `prism.css` to `prism-onedark.css`
